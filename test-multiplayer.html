Multiplayer Connection Test

Test 1: Local Testing

Test multiplayer in the same browser with two tabs.

  1. Open your game in a new tab
  2. Host a game and copy the Game ID
  3. Open another tab with your game
  4. Join the game using the copied ID
  5. Test playing moves between tabs
Expected Result: Moves should sync between tabs instantly.

Test 2: Cross-Device Testing

Test multiplayer between different devices.

  1. Deploy your game to GitHub Pages
  2. Open the game on two different devices
  3. Host a game on one device
  4. Join the game on the other device
  5. Test playing moves between devices
Expected Result: Moves should sync between devices instantly.

Test 3: Connection Issues

Test what happens when connection fails.

  1. Try joining with an invalid Game ID
  2. Disconnect one player during the game
  3. Test with slow internet connection
  4. Try from different network types
Expected Result: Should show appropriate error messages.

Test 4: Browser Compatibility

Test with different browsers.

  1. Test with Chrome/Edge
  2. Test with Firefox
  3. Test with Safari
  4. Test on mobile browsers
Expected Result: Should work on all modern browsers.

Troubleshooting Tips

  • • If connection fails, try refreshing both players' pages
  • • Make sure both players have stable internet connections
  • • Try disabling VPN if you're using one
  • • Check browser console for error messages
  • • Some corporate networks may block WebRTC connections
